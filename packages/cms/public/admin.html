API Endpoints

Use these REST API endpoints to manage content programmatically:

Psychoactives

GET /api/psychoactives - List all psychoactives

GET /api/psychoactives/:slug - Get single psychoactive

Combos

GET /api/combos - List all combos

GET /api/combos/:slug - Get single combo

Risks

GET /api/risks - List all risks

GET /api/risks/:drug1/:drug2 - Get risk for combination

Health

GET /health - API health check

For full API documentation and content editing instructions, see ADMIN_GUIDE.md in the CMS package directory.
